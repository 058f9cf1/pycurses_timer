# Pycurses Timer

A simple timer built with pycurses.

## Installation

Clone the repository with `https://github.com/058f9cf1/pycurses_timer.git`.

## Usage

```
./curses_timer.py <time>
```
Time should be given in seconds. Press `q` to quit.
